# Saturn Browser

The Saturn browser is simple, fast and made in Visual Basic. This GitHub repository contains the source code and official downloads. To compile it, you will need:
- Visual Studio 2019.
- .NET Framework 4.6.2
- Visual C++ Redistributable for Visual Studio 2019
- any installed Flash ActiveX on your system.

Saturn Browser uses the following fonts:
- Century Gothic
- Segoe MDL2 Assets
- Hololens MDL2 Assets
